## My pros

**I focus on learning solid foundations.** The web is based on HTTP requests. Most programs write to stdout. Bubble sort always works in the same way. When I know the basics of something, then I can spin it around and break it apart more effectively.

**I'm aware of the fact that I still have much to learn.** With every new project I start I try to learn something new. Dealing with errors is part of the process. Fail faster to learn quicker.

**I choose the right tools for the job.** If the project I'm working on is simple and doesn't have any specific requirements, then I'm going to work with the simplest possible stack. Simple projects should be built in a simple way.

**I'm a Linux user.** I have been using Linux as the only operating system on my personal computer for almost ten years now. I cannot work without having a TTY interface nearby.

**I'm an "old school", no bells-and-whistles kinda developer.** Give me a text editor and I will be able to program anything. I have a hard time using modern applications like Figma.

**I do not use AI tools.** I am not interested in working on projects involving the use of AI either. I believe that technology should strive to be sustainable and ethical. Instead of using AI assistants, I focus on learning skills that I will be able to use in any scenario.
